Brain Boost and Tracker App

This project was part of E-health: Promoting Healthy Ageing course at UPM where we had to create a high fidelity prototype for an interface that Promoting Healthy Ageing by Cognitive Exercise for Frailty users. So we developed an app called Brain Boost which is an iPad game application for the elderly for performing cognitive exercises. The tracker app is used by the caretaker to give personalised rewards to the user. The features of the app are mentioned in detail on the report. The high-fidelity prototype was created using Sketch and Invision App (link to invision app attached).
